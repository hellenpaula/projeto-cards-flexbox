##  📌 Sobre:

Este é um projeto de interface desenvolvido com HTML e CSS, com foco na construção de layouts utilizando Flexbox.

A proposta do projeto é apresentar destinos turísticos naturais ao redor do mundo, destacando suas principais características e criando uma experiência visual agradável através de cards interativos.

Durante o desenvolvimento, foram trabalhados conceitos importantes como organização de layout com flexbox, alinhamento de elementos, espaçamento, estilização de componentes e efeitos visuais com hover.

## 🧩 Funcionalidades:

- Exibição de cards com destinos turísticos.

- Estrutura organizada com Flexbox.

- Cards contendo:

  - Imagem do destino

  - Localização da imagem

  - Título

  - Descrição

  - Botão com link externo

- Efeito de hover nos cards com:

  - Leve aumento de escala

  - Alteração de sombra e borda

  - Botão interativo com mudança de cor ao passar o mouse.

  - Layout adaptável com flex-wrap, permitindo melhor distribuição em telas menores.

  
## 💻 Tecnologias utilizadas:

## ⚙️ Estrutura e organização do layout:

O projeto é dividido em duas partes principais:

### 1️⃣ Cabeçalho (título da página)

- Apresenta o tema do projeto: Destinos turísticos - Natureza.

- Uso de destaque visual com cor para reforçar o conceito.

### 2️⃣ Container de cards

- Utiliza display: flex para organizar os elementos.

- Propriedades utilizadas:

  - justify-content: center

  - align-items: center

  - gap

  - flex-wrap

- Cada card é estruturado em:

  - Área da imagem (area_img)

  - Área de texto (area_texto)

  - Botão de ação (btn)
